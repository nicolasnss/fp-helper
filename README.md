# FP-Helper - Calculadora de Pontos de Função

O **FP-Helper** é uma ferramenta de automação para métricas de software baseada no método de Análise de Pontos de Função (APF). Este projeto foi desenvolvido como produção técnica para o mestrado em **Cybersecurity** da **Universidade de Brasília (UnB)**.

## 🚀 Tecnologias
* **Linguagem:** Java 17
* **Framework:** Spring Boot 3.5.11
* **Gerenciamento:** Maven
* **Segurança:** Algoritmo de Integridade SHA-512

## 🛠️ Como Executar
1. Clone o repositório: `git clone https://github.com/nicolasnss/fp-helper.git`
2. Navegue até a pasta: `cd fp-helper`
3. Execute o projeto: `.\mvnw.cmd spring-boot:run`
